Postupak za setupovanje:

1. u folderu gde se nalazi package-lock.json pokrenuti cmd i pokrenuti naredbu $npm install
2. u folderu gde se nalazi package-lock.json pokrenuti cmd i pokrenuti naredbu $npm install
3. upaliti xampp server i pokrenuti apache i mysql servise
4. u bazi (http://localhost/phpmyadmin) dodati bazu pod imenom: evidencija_satnica
5. u bazi evidencija_satnica importovati sadrzaj fajla bazaPodataka/evidencija_satnica.sql
6. pokrenuti app.js (preko commandline ili nekog drugog interfejsa)
7. pokrenuti Build/EvidencijaSatnica.exe
