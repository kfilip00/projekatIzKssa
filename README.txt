Postupak za setupovanje:

1. u folderu gde se nalazi package.json pokrenuti cmd i pokrenuti naredbu $npm install
2. upaliti xampp server i pokrenuti apache i mysql servise
3. u bazi (http://localhost/phpmyadmin) dodati bazu pod imenom: evidencija_satnica
4. u bazi evidencija_satnica importovati sadrzaj fajla bazaPodataka/evidencija_satnica.sql
5. pokrenuti app.js (preko commandline ili nekog drugog interfejsa)
6. pokrenuti Build/EvidencijaSatnica.exe
