Postupak za setupovanje:

1. upaliti xampp server i pokrenuti apache i mysql servise
2. u bazi (http://localhost/phpmyadmin) dodati bazu pod imenom: evidencija_satnica
3. u bazi evidencija_satnica importovati sadrzaj fajla bazaPodataka/evidencija_satnica.sql
4. pokrenuti app.js (preko commandline ili nekog drugog interfejsa)
5. pokrenuti Build/EvidencijaSatnica.exe
